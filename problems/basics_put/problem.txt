`{bold}level{/bold}` を使ってLevelDBデータストアをオープンするプログラムを書いてみましょう。

データストアへの絶対パスはプログラムの最初のコマンドライン引数に渡されます。

二つ目のコマンドライン引数はJSONエンコードされたオブジェクトを含みます。
このオブジェクトをパースして、それぞれのプロパティを {bold}put{/bold} を
利用してデータストアに書き込んでください。プロパティのキーはデータストアのキーに、
値はデータストアの値としてください。

検証にはデータストアの読み込みと、それによって読み込まれるオブジェクトに対して行われます。

----------------------------------------------------------------------
ヒント:

LevelUPの `put()` メソッドはとてもシンプルです。キーと値を指定するだけです。
オプションのコールバックはデータがストアにコミットされたことを検出したい場合、
もしくはエラーが返って来てないかを検証する際に利用します。コールバックがなく、
エラーが発生した場合にはそのエラーは Throw されます。

  var db = level('/path/to/db/')
  db.put('foo', 'bar', function (err) {
    if (err)
      return console.error('Error in put():', err)
    console.error('put foo = bar')
  })

全てのコールバックが返ってきた後に `db.close()` を呼んでも良いですが、
これはオプションです。ストアの操作終了時にはデータストアは自動的にクローズされます。
データストアを開くことでプログラムが永遠に走り続けることはありません。

`levelmeup run program.js` を走らせた時にデバッグ出力を見たい場合、
console.log ではなく、console.error を使ってください。
